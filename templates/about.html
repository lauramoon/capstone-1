About

This website is a capstone project for Springboard's software engineering curriculum. It is built with Flask, SQLAlchemy and a PostgreSQL database.

The plant data comes from Trefle, a global plants API. Accuracy of the data is not guaranteed. Plant image copyrights remain with the original copyright holder.

The site uses Heroku's free hosting and database options, so load time may be long. Accounts may be deleted without notice for any reason, including limited storage space.

This site collects no personal data. There is no way to recover a lost password.
